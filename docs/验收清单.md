## 人工验收特别关注

- 主窗口、四类技术域和评分页是否保持紧凑桌面布局。
- 下拉框悬停滚轮是否不会误切换选项。
- 日历弹层、下拉弹层和临时提示是否绑定父窗口，避免出现标题为 `python` 的临时窗口。
- 保存失败是否至少记录日志并给出可见失败提示；自动保存成功是否保持静默。
- 打包后首次运行的数据文件是否位于当前用户可写应用数据目录。
- 导出的 Excel 文件在 Microsoft Excel/WPS 中打开后，sheet 名称、列宽、合并单元格、颜色和空值表达是否符合需求文档。
